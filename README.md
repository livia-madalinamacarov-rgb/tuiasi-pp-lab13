# Lab 13 — Python Programare Funcțională

Template GitHub Classroom pentru laboratorul 13 de Programare Python.

## Conținut

- **`lab13/functional_ops.py`** — Operații funcționale pe dict cu map/filter/lambda (stub de implementat)
- **`lab13/html_combiner.py`** — Combinator de fișiere HTML (stub)
- **`lab13/main.py`** — Entry point demonstrativ
- **`tests/test_lab13.py`** — Suite de teste (nu modifica)

## Cum se rulează

```bash
# Rulare teste
uv run pytest

# Rulare cu output detaliat
uv run pytest -v

# Demonstrație
uv run python -m lab13.main
```

## Cum se instalează dependențele

```bash
uv sync
```

## Cerințe

- Python >= 3.11
- uv (package manager)
